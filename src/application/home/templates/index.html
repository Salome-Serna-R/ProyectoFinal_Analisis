Capítulos y Métodos Numéricos

Explora la implementación, comparación y análisis de diversos métodos numéricos organizados en capítulos, con interfaces que facilitan la entrada de datos, visualización gráfica y generación automática de informes de resultados para ayudarte a elegir el mejor método según el problema.

Capítulo 1: Solución de ecuaciones no lineales

Se entregan los métodos de bisección, regla falsa, punto fijo, Newton-Raphson, secante y al menos uno de raíces múltiples. La interfaz permite ingresar funciones algebraicas de manera flexible, con ayuda para calcular derivadas y explicaciones claras sobre el ingreso de datos. La solución se presenta mediante gráficas y tablas de iteraciones. Además, se genera un informe automático de ejecución y comparación de todos los métodos aplicados a un problema específico, identificando el método más eficiente. El usuario puede decidir si desea generar este informe.

Capítulo 2: Solución de sistemas de ecuaciones lineales

Se entregan los métodos de Jacobi, Gauss-Seidel y Sobre-relajación sucesiva (SOR). La interfaz muestra tablas con las soluciones iterativas, calcula y muestra el radio espectral, e indica si cada método converge o no según dicho radio. Se permite ingresar matrices hasta tamaño 7x7 con explicaciones claras para la entrada de datos. Al igual que en el capítulo 1, se genera un informe automático y comparativo de los métodos aplicado a un problema específico, resaltando el método más efectivo, con opción para que el usuario decida si ejecutar el informe.

Capítulo 3: Interpolación polinómica

Se entregan los métodos de Vandermonde, Newton interpolante, Lagrange, spline lineal y spline cúbico. La interfaz permite ingresar hasta 8 puntos de datos con explicaciones sencillas. Los resultados se muestran gráficamente junto con el polinomio solución y su tabla de valores. Se genera un informe automático y comparativo que evalúa el rendimiento de cada método ante un problema específico, identificando el más adecuado. El usuario decide si desea ejecutar el informe.
